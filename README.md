# 공통테스트-전수진

## 실행방법

```
git clone https://github.com/soozynn/coding-test_stayfolio.git

```

```
npm install
```

설치 후,

```
npm test
```

테스트가 무사히 통과하는지 확인해주세요!

## 구조

- lib<br/>
  lib 폴더에는 공통 테스트 파일들이 들어있습니다.

- test<br/>
  lib 폴더 내부의 테스트 파일들이 잘 통과하는지 확인하는 테스트 파일들이 들어있습니다.
